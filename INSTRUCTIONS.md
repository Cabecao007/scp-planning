# 📋 Guide complet — Mettre le planning sur GitHub

## Étape 1 — Créer un compte GitHub (si pas déjà fait)

1. Aller sur **https://github.com**
2. Cliquer **"Sign up"**
3. Choisir un nom d'utilisateur (ex: `scp-houppeville`)
4. Entrer votre email et un mot de passe
5. Vérifier votre email

---

## Étape 2 — Créer un nouveau dépôt (repository)

1. Une fois connecté, cliquer sur le **"+"** en haut à droite
2. Choisir **"New repository"**
3. Remplir :
   - **Repository name** : `scp-planning`
   - **Description** : `Planning infirmières SCP Houppeville`
   - Cocher **"Public"** ← important pour GitHub Pages
   - Cocher **"Add a README file"**
4. Cliquer **"Create repository"**

---

## Étape 3 — Uploader les fichiers

1. Dans votre nouveau dépôt, cliquer **"Add file"** → **"Upload files"**
2. Glisser-déposer ces 3 fichiers :
   - `index.html`
   - `README.md`
   - `.gitignore`
3. En bas, écrire dans "Commit changes" : `Ajout planning SCP 2027`
4. Cliquer **"Commit changes"**

---

## Étape 4 — Activer GitHub Pages (pour avoir un lien web)

1. Dans votre dépôt, aller dans **"Settings"** (onglet en haut)
2. Dans le menu gauche, cliquer **"Pages"**
3. Sous "Branch", choisir **"main"** et **"/ (root)"**
4. Cliquer **"Save"**
5. ⏳ Attendre 2-3 minutes
6. L'adresse apparaît : `https://[votre-username].github.io/scp-planning`

---

## Étape 5 — Accéder au planning

✅ **Sur PC/Mac** : ouvrir le lien dans Chrome ou Firefox  
✅ **Sur iPhone** : ouvrir dans Safari → ⬆ Partager → "Sur l'écran d'accueil"  
✅ **Sur Android** : ouvrir dans Chrome → bannière d'installation automatique

---

## 🔄 Mettre à jour le planning plus tard

1. Aller sur votre dépôt GitHub
2. Cliquer sur `index.html`
3. Cliquer l'icône ✏️ (crayon) pour éditer
4. Ou cliquer **"Add file"** → **"Upload files"** pour remplacer le fichier
5. Le site se met à jour automatiquement en quelques minutes

---

## 💡 Conseils

- **Sauvegarder vos données** : utilisez le bouton "💾 Exporter" régulièrement
  pour garder une copie JSON de vos modifications et vacances
- **Partager le lien** avec les infirmières une fois le site en ligne
- Si vous changez d'ordinateur : importez votre JSON avec "📥 Importer"

---

*En cas de problème : toutes les données restent dans le navigateur local,
le fichier JSON exporté est votre sauvegarde principale.*
